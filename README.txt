A Python application to predict which Xbox game a visitor will be most interested in based on their search query. 
See http://www.kaggle.com/c/acm-sf-chapter-hackathon-small for description and data.


correct.py - a module with spelling utilities
LatentSemanticIndexer.py - a module which applies the Latent Semantic Indexing
transformation to a set of documents
PorterStemmer.py - an implementation of the Porter stemming algorithm
PredictXboxGame.py - Predict which Xbox game a visitor will be most interested in based on their search query. 
VectorSpaces.py - Create a vector space model for a set of documents
